# Details der lokalen Lastausgleichsfunktion anzeigen

Alle Interaktionen mit einer lokalen Lastausgleichsfunktion erfolgen über die entsprechende Detailseite, auf der Sie Servicegruppen verwalten, Statistikdaten anzeigen und den SSL-Auslagerungsstatus überprüfen können. Führen Sie die nachfolgenden Schritte aus, um eine Liste lokaler Lastausgleichsfunktionen anzuzeigen und auf die Seite einer bestimmten Lastausgleichsfunktion zuzugreifen:

1. Öffnen Sie das [Kundenportal ![Symbol für externen Link](../../icons/launch-glyph.svg "Symbol für externen Link")](https://control.softlayer.com/){: new_window} über einen Browser und melden Sie sich bei Ihrem Konto an.
2. Wählen Sie im Navigationsbereich des Kundenportals **Netz > Lastausgleich > Lokal** aus.
3. Klicken Sie auf der Listenseite **Lastausgleichsfunktionen** auf die Registerkarte **Lokale Lastausgleichsfunktionen**, um die Liste der lokalen Lastausgleichsfunktionen anzuzeigen.
4. Klicken Sie auf den Link **VIP-Adresse**, um auf die Seite mit den **Details** zuzugreifen. Alternativ können Sie im Dropdown-Menü der Zeile, die der zu verwaltenden lokalen Lastausgleichsfunktion entspricht, auf **Aktionen > Details anzeigen** klicken.

In der Anzeige mit den **Details der lokalen Lastausgleichsfunkion** ist standardmäßig die Registerkarte 'Servicegruppen' ausgewählt, auf der Benutzer Servicegruppen hinzufügen, löschen oder bearbeiten, Verbindungen zurücksetzen oder Services hinzufügen können. Auf der Registerkarte 'Statistiken' können Statistikdaten für die einzelnen Servicegruppen angezeigt werden und auf der Registerkarte für die SSL-Auslagerung können [SSL-Auslagerungen konfiguriert](configure-ssl-offloading-load-balancer.html) werden.
